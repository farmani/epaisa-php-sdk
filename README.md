## Install

Via Composer

``` bash
$ composer require eigitallabs/epaisa-php-sdk
```

## Usage

``` php
$skeleton = new eigitallabs\epaisa-php-sdk();
echo $skeleton->echoPhrase('Hello, League!');
```
